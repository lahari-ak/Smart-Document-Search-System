## Overview:
Smart Document Search and Question Answering System is an AI-powered web application that allows users to upload PDF documents and ask questions in natural language. The system uses semantic search and AI-based retrieval to generate accurate answers from the uploaded documents.
## Features:
* Upload and process PDF documents
* AI-powered question answering
* Semantic search for relevant information retrieval
* Source-based answer generation
* User authentication and document management
* Chat history and backup support
* Light Mode and Dark Mode
* Modern UI with 3D hover effects
* Backend connection validation
## Tech Stack:
Frontend: HTML, CSS, JavaScript
Backend: FastAPI, Python
AI Integration: API Key-based AI model, Vector Embeddings, Semantic Search
Workflow:
1. User uploads a PDF document.
2. The document is processed and indexed.
3. Backend connection is verified.
4. User asks questions about the document.
5. The system retrieves relevant content and generates accurate answers.
6. Chat history is saved for future reference.
